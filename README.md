Quando abrirem o VS, vão à secção de Extensões e instalem uma extensão chamada 'Live Server'.

<img width="1920" height="1080" alt="2026-03-05_15-45-56" src="https://github.com/user-attachments/assets/be6ba54e-6051-44f8-8860-db8adcb241a8" />

De seguida vai aparecer no canto inferior direito um botão que diz 'Go live'.

<img width="1920" height="1080" alt="2026-03-05_15-49-44" src="https://github.com/user-attachments/assets/44bc12f7-2153-4dcb-abdc-700f8843e597" />
<img width="1920" height="1080" alt="2026-03-05_15-46-30" src="https://github.com/user-attachments/assets/b821e8c1-496a-4ceb-aacb-a959088a0b6a" />

Quando clicarem, vai abrir uma nova aba no vosso browser predefinido com o programa a funcionar.

Caso queiram desligar o server clicam outra vez onde carregaram inicialmente para abrirem o server, que agora há de dizer algo como 'Port: 5500' (Eu acho que o número da porta varia), e o servidor desliga-se.
